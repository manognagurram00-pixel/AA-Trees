# AA Tree Implementation

A C++ implementation of the AA Tree self-balancing binary search tree, supporting efficient insertion, deletion, search, and traversal operations while maintaining balanced tree properties using skew and split operations.

## Features
- AA Tree implementation from scratch
- Insertion, deletion, and search operations
- Tree traversals (Inorder, Preorder, Postorder)
- Automatic balancing using Skew and Split operations
- Console-based visualization of tree updates

## Tech Stack
- C++
- STL

## Getting Started

1. Clone the repository
   ```bash
   git clone <repository-url>
   ```

2. Compile the project
   ```bash
   g++ main.cpp AATree.cpp -o aa_tree
   ```

3. Run the executable
   ```bash
   ./aa_tree
   ```
